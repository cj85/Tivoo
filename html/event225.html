Summary and Details

SSRI~ Workshop: Using LaTeX

Start Time: 2011-09-29 03:00:00
End Time: 2011-09-29 05:00:00
Summary: LaTeX is becoming the most widely used document processing system in the social sciences with good reason: it generates publication quality documents, allows for precise mathematical notation and equations, has a built in citation manager and it is FREE. This workshop will cover how to build a basic document with an abstract, tables, figures and citations. No previous knowledge of LaTeX or programming is needed.
